# Contributing

Contributions are welcome! Since this package is still under development, please discuss
any changes via GitHub Issues before starting work.

## Setting up a development environment

```bash
# Install development dependencies
make init

# Run tests
make test

# Format code
make format

# Run all linting checks
make ci-lint
```

## Development notes

- Code is formatted with Black and isort
- Linting via flake8
- Type checking via mypy
- Tests use pytest
